Three New HTML Tags

Tag 1: Details tag (< details > < /details >)

The main purpose of this tag is to allow people to hide or show information on demand. If you click the triangle on the left side of the webpage, you will reveal more information that is hidden from you by default.

This tag is very useful to keep webpages clean without all of this text being shown in front of you at once.

To use this tag, you have to type it into the HTML body and then within the tag, you can write a summary tag, which is essentially the title for the details, and after that comes a paragraph tag, which you can type just like any other paragraph tag. Like other tags, detail tags could also be used with CSS to change colors.

An example of code for a details tag:

< details >
 < summary>Summary of details< /summary>
 < p> This information is hidden by default< /p>
< /details>

I got information about this tag from W3 Schools, which is a very good source for help with HTML and its tags.

Tag 2: Address tag (< address>< /address>)

The main purpose of the address tag is for people to display contact information anywhere in their webpage. An example with my contact information could be seen below.

Email: np0481059@student.luzerne.edu
Cell Phone: (570)-555-4716

The code used to display this information:

< address>
 Email: np0481059@student.luzerne.edu< br>
 Cell Phone: (570)-555-4716
< /address>

The text is light blue because, like the details tag, this tag could also be used with CSS to change colors.. However, the contact information is also italicized by the tag itself so that people know that it is contact information."

I got information about this tag from the MDN Web Docs from Mozilla, a company well-known for their Firefox web browser.

Tag 3: Video tag (< video>< /video>)

Being introduced with HTML5 in 2008, this tag allows people to view videos on webpages without having to go directly to the video's link.

For example, below you can see a beautiful blue sky and some trees.


This could be done with the following line of code:

< video width="640" height="480" src="https://ia600701.us.archive.org/11/items/Flickr-5971166253/sky-5971166253.mp4" controls>< /video>


This video is played in a resolution of 640x480 as indicated in the code. You could switch to full screen, set the playback speed, and even download the video to your own device. Usually, you could also change the volume, but since this video does not have audio, it is greyed out.

This is all because of the 'controls' command being used at the end of the tag. Without this command, the video would only be shown as a still image with a 640x480 resolution. In order for this tag to work, it also has to have a direct link to a video in the anchor tag, which means that YouTube videos cannot be used. There is a more advanced embed tag for those videos.

I got information about this tag from HTML.com, another great website for learning HTML.
