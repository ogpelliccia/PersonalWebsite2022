HTML Help

Creating webpages with HTML is mostly new to me. A few years back, I have tried to learn HTML using WikiHow , but I didn't get too much into it. However, now, I have taken CIS-145 at LCCC to learn how to make webpages. I would like to share with you some websites that have helped me get this far in web development.

W3Schools.com This is a very useful website filled with tutorials for countless web and programming languages. Such languages include HTML (of course), CSS, Java, JavaScript, PHP, Python, SQL, and C++.
HTML.com Unlike W3Schools, this website is entirely focued on HTML and CSS, hence its name, and has plenty of tutorials and information about tags and attributes.
Mozilla Developer Resources From the same people who developed the Firefox web browser, this is a great resource for references and guides for HTML, CSS, JavaScript, and Web APIs. From certain tags and elements to whole tutorials and guides, this page has you covered.
